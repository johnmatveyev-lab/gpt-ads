# Design Direction

## Creative Brief

Design a premium conversion website for local businesses that want to explore ChatGPT Ads early. The visual language should feel modern, sharp, AI-native, and trustworthy without pretending to be OpenAI or using unverified proof.

## Visual System

### Colors

Use a dark interface with mint/green accents.

- Background: `#05080c`
- Background elevated: `#0b1117`
- Panel dark: `#111820`
- Panel light: `#f7faf8`
- Text primary dark section: `#f8fbff`
- Text secondary dark section: `#b7c1c9`
- Text primary light section: `#071018`
- Mint: `#5ff3b0`
- Green: `#20c986`
- Deep teal: `#0b7c68`
- Border dark: `rgba(255,255,255,0.14)`
- Border mint: `rgba(95,243,176,0.45)`

### Typography

- Hero should be large, uppercase or near-uppercase, tightly composed, and responsive.
- Body copy should be clear, generous, and readable.
- Do not scale fonts directly with viewport width.
- Avoid negative letter spacing.
- Keep compact panels at modest heading sizes.

### Components

Required components:

- Top navigation.
- Hero badge.
- Primary CTA button.
- Secondary text/link CTA.
- Benefit icon row.
- Phone/ad preview mockup.
- Ava consultation card.
- Four-step process cards.
- Local-business solution cards.
- FAQ accordion.
- Trust/proof bar using verified proof only.
- Footer disclaimer.
- Sticky chat launcher.

### Layout

Desktop:

- Hero should use a two-column composition with text left and visual/agent panel right.
- First viewport should hint at the next section.
- Process section should be light or high-contrast to break the dark hero.

Mobile:

- Stack hero content before mockup.
- Keep CTA visible above fold.
- Process cards stack vertically.
- Agent card should remain easy to tap.
- No text should overlap icons, cards, or mockups.

## Iconography

Use lucide icons where possible:

- `MessageCircle` for Ava/chat.
- `Target` for targeting/fit.
- `TrendingUp` for growth.
- `ShieldCheck` for compliance/trust.
- `CalendarCheck` for booking.
- `Users` for audience.
- `ChartNoAxesCombined` or `BarChart3` for performance.
- `ArrowRight` for CTA motion.

## Motion

Motion should be subtle and premium:

- Soft gradient glow.
- Slow background line movement.
- Button hover lift.
- Card hover border glow.
- Chat launcher pulse only if not distracting.

Respect `prefers-reduced-motion`.

## Proof Policy

Design may include placeholder proof blocks during development, but production must hide or replace:

- Third-party logos.
- Official partner badges.
- Campaign count.
- Managed spend.
- Retention numbers.
- Testimonials.
- Case studies.

If proof is unavailable, use:

- "Built for early ChatGPT Ads readiness."
- "Designed for local-business launch planning."
- "Policy-aware setup and measurement planning."
